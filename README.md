# 🚀 Active Directory Home Lab Journey

Welcome to my Active Directory Home Lab — a personal project built from curiosity, perseverance, and my passion for growing in the IT field. This repository is more than just a lab setup; it represents my hands-on journey into system administration, networking, troubleshooting, and real-world IT environments.

Through this project, I am transforming theory into practical experience by building a complete Windows Server environment using VirtualBox, client machines, and Active Directory services. Every configuration, error, troubleshooting step, and success in this lab helps me strengthen my skills and deepen my understanding of how enterprise systems work behind the scenes.

## 🔧 What I’m Working On

### 🖥️ Setup Simple Configuration

Building the foundation of the lab environment by configuring virtual machines, networking, and system communication.

### 🌐 Windows Server Installation & Configuration

Installing and preparing Windows Server to simulate a real-world enterprise infrastructure.

### 👥 Active Directory Domain Services (AD DS)

Creating domains, users, groups, and organizational units while learning identity and access management.

### 💻 Client Machine Integration

Connecting Windows client machines to the domain to simulate an organizational environment.

### 📡 DHCP & DNS Configuration

Configuring essential networking services that allow devices and systems to communicate properly.

### 🔒 Group Policy Management

Applying policies and restrictions to manage users and computers like in professional IT environments.

### 📂 Shared Folders & Permissions

Managing file sharing, access control, and permissions to improve security and organization.

### 🧠 Troubleshooting & Documentation

Documenting issues, solutions, and lessons learned throughout the lab to improve problem-solving skills.

### 🧪 VirtualBox Networking

Exploring NAT, Internal Networks, and virtual communication between systems inside the lab environment.

## 🎯 Goal of This Project

My goal is to gain real hands-on experience in IT Support, System Administration, and Networking while continuously improving my technical and troubleshooting skills. This repository serves as both my learning journal and portfolio as I work toward becoming a future System Administrator and Cloud Engineer.

Every lab completed is another step closer to my dream career in IT.
