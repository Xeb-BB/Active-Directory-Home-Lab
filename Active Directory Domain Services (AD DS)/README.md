# Lab Module: Active Directory Domain Services (AD DS)
## Introduction
### Active Directory Domain Services (AD DS) is the foundation of enterprise identity and access management (IAM). This lab covers promoting Windows Server 2022 to a Domain Controller, establishing the css.com root domain, and structuring users, groups, and Organizational Units (OUs) for centralized network authentication.

## Tools & Technologies Used
### Windows Server 2022 (Desktop Experience)

### AD DS Role & Promotion Wizard

### Active Directory Users and Computers (dsa.msc)

### IAM Objects: Organizational Units, Security Groups, and User Accounts

## What to Expect in This Section
### Server Promotion: Activating AD DS and provisioning the css.com forest.

### OU Structure: Designing a logical directory layout to organize network assets.

### Account Provisioning: Creating secure domain user profiles (ex. jbenriquez).

### Group Management: Nesting users into security groups for scalable access control.
