# 🌐 Setup Network Configuration

This project is a simple networking lab using VirtualBox for hands-on practice in IT Support and System Administration 💻

In this lab, I will:

* 🖥️ Setup virtual machines
* 🌍 Configure basic network settings
* 🔌 Test server and client connectivity
* 📡 Assign IP addresses
* 🛠️ Prepare the environment for Active Directory

### 🧰 Tools Used

* VirtualBox
* Windows Server 2022 (Server)
* Windows 10 PRO (Client)
* Basic Networking

### 📂 What to Expect in this Repository

* ✅ Procedure Checklist
* 🗺️ Network Flowchart
* 📸 Lab Screenshots
* 📝 Configuration Documentation
* 🛠️ Troubleshooting Notes

### 🎯 Goal

To improve my networking, troubleshooting, and virtualization skills through real-world lab practice and documentation.
