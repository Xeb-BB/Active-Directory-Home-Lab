# Lab Module: DHCP & DNS Configuration
## Introduction
This lab module focuses on deploying the network baseline required for any enterprise environment. By configuring Domain Name System (DNS) and Dynamic Host Configuration Protocol (DHCP) services on Windows Server 2022, you will establish reliable local name resolution and automated IP address management. This ensures that every device on the domain can communicate seamlessly without manual networking conflicts.

## Tools & Technologies Used
### Windows Server 2022 (Configured as a Domain Controller)

### DNS Manager (Forward and Reverse Lookup Zones, PTR Records)

### DHCP Manager (Post-installation Authorization, Active Directory Integration)

### Command Line Interface: nslookup for network resolution verification

## What to Expect in This Section
### A streamlined workflow to install and authorize network roles simultaneously.

### Step-by-step guidance on creating a Reverse Lookup Zone and an automated Pointer (PTR) record to map IPs back to hostnames.

### A walkthrough for setting up a dedicated DHCP Scope (Client_LAN), defining IP pools, and excluding critical static server addresses.

### Practical verification steps using client-side diagnostic commands.
